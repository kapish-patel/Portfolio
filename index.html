Featured Projects

Drop box Cloud clone

   

During my university years, I engineered a Dropbox-inspired cloud storage platform capable of housing up to 1.5TB of data. Leveraging scrap bare metal recycling components, I constructed a server for this purpose. The platform facilitated easy data sharing among friends, though its discontinuation resulted from network security concerns raised by the university IT department. Nevertheless, the project provided invaluable insights into both cloud storage technology and sustainable resource utilization.

Check it out!
